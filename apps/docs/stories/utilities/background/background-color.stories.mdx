<Lead>Select from a range of colors for the background of an element.</Lead>

## Classes

Background colors can be set with the CSS class `bg-***` for example `bg-green`.

Every color of the [Design Tokens](?path=/docs/tokens-overview--documentation) can be used like that.

<CssBackgroundColors />

## Basic

<Code
  language="html"
  code={`
<div class="bg-green p-normal">Green background</div>
`}
/>

## Inverted Colors

For every color, an inverted variant is defined to ensure optimal readability when combining background and font colors.

<Code
  language="html"
  code={`
<div class="bg-primary p-normal">
  <p class="text-white">Hello World</p>
</div>
`}
/>

## Pseudo States

States such as Focus, Hover, and Active can be utilized by applying the respective prefixes: **focus:**, **hover:**, and **active:**.

<Code
  language="html"
  code={`
<div class="bg-green-3 hover:bg-green-4 active:bg-red-3 cursor-pointer">
  <p class="text-white p-normal">Hover and click me!</p>
</div>
`}
/>
