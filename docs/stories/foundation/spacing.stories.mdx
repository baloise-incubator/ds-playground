## Spacing Sizes

There are 10 spacings across the Design System.

<TokensSpacing />

## Spacing Guidelines

### Heading + Content

There are two sizes of spaces after a Heading to its own section 16px or 8px.

import SpacingHeadings from '../assets/images/foundations/spacing/heading-content.svg'

<ImageCanvas size="large">
  <img loading="lazy" src={SpacingHeadings} />
</ImageCanvas>

Sections are divided with 128px, however if the section is connected use 64px instead.

import Sections from '../assets/images/foundations/spacing/sections.svg'

<ImageCanvas size="large">
  <img loading="lazy" src={Sections} />
</ImageCanvas>

### Cards

To use multiple cards it is recommended to use the grid to get the correct spacings. The space between cards is 32px.

import CardLayout from '../assets/images/foundations/spacing/card-layout.svg'

<ImageCanvas size="large">
  <img loading="lazy" src={CardLayout} />
</ImageCanvas>
