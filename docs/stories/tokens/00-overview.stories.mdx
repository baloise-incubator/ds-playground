## What are Design Tokens

A design token is a standardized, reusable unit of design information that encapsulates visual
and stylistic elements, ensuring consistency across various applications and platforms.
It serves as a central reference point for maintaining and implementing design decisions.

## Use in code

The tokens are exported into CSS Variables, Sass, Less and JavaScript.

Here are two examples how to use them in a web application.

```html
<div style="background: var(--bal-background-primary)"></div>
```

```css
.my-block {
  background: var(--bal-background-primary);
}
```

## Figma

To integrate design tokens into your Figma file, employ the [Style Dictionary to Variables](https://www.figma.com/community/plugin/1254182310197842774/style-dictionary-to-variables) plugin.
