## Basic

The grid is based on 12 grid. On mobile the grid will use the full width.

<Code
  language="html"
  code={`
<div class="grid">
    <div class="col">
      <div class="bg-green p-small radius-normal">First column</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">Second column</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">Third column</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">Fourth column</div>
    </div>
  </div>
`}
/>

## Rows

By adding the CSS class `is-multiline` the grid can be shown on more than one row.

<Code
  language="html"
  code={`
<div class="grid is-multiline">
  <div class="col is-full">
    <div class="bg-green p-small radius-normal">First row</div>
  </div>
  <div class="col is-full">
    <div class="bg-red-3 p-small radius-normal">Second row</div>
  </div>
</div>
`}
/>

## Nested

Grids can be nested to have more flexibility in your structure.

Just keep the structure grid > column > grid > column > ...

<Code
  language="html"
  code={`
<div class="grid">
  <div class="col is-half">
    <div class="bg-green p-small radius-normal">Half</div>
  </div>
  <div class="col is-half">
    <div class="grid">
      <div class="col is-half">
        <div class="bg-red-3 p-small radius-normal">Half</div>
      </div>
      <div class="col is-half">
        <div class="bg-red-3 p-small radius-normal">Half</div>
      </div>
    </div>
  </div>
</div>
`}
/>

## Space

By default add a gap `var(--bal-column-gap)` of 1rem between the grid.
This can be avoided with setting the CSS class `is-gapless`.

<Code
  language="html"
  code={`
<div class="grid is-gapless">
    <div class="col">
      <div class="bg-green p-small radius-normal">First column</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">Second column</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">Third column</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">Fourth column</div>
    </div>
  </div>
`}
/>

## Vertical Alignment

To center the grid of the grid vertically set `is-vcentered` CSS class.

<Code
  language="html"
  code={`
<div class="grid is-vcentered">
    <div class="col is-two-thirds">
      <div class="bg-green-2 p-small radius-normal">First column</div>
    </div>
    <div class="col">
      <div class="bg-green-2 p-small radius-normal">Second column with some random large text to make a good case here. Unfortunately, more content is needed to make it visible that the first column is now vertically centered.</div>
    </div>
  </div>
`}
/>

## Horizontal Alignment

To center the grid of the grid set `is-centered` CSS class.

<Code
  language="html"
  code={`
<div class="grid is-centered">
    <div class="col is-one-third">
      <div class="bg-green-2 p-small radius-normal">My centered column</div>
    </div>
  </div>
`}
/>

## Stretch

To stretch the inner element of a column use the CSS class `h-full`.

<Code
  language="html"
  code={`
<div class="grid is-multiline">
    <div class="col is-two-thirds">
      <div class="bg-green-2 p-small radius-normal h-full">Stretches to the full height of the row</div>
    </div>
    <div class="col is-one-third">
      <div class="bg-green-2 p-small radius-normal h-full">Second column with some random large text to make a good case here. Unfortunately, more content is needed to make it visible that the first column is now vertically centered.</div>
    </div>
    <div class="col is-half">
      <div class="bg-green-2 p-small radius-normal h-full">Second column with some random large text to make a good case here. Unfortunately, more content is needed to make it visible that the first column is now vertically centered.</div>
    </div>
    <div class="col is-half">
      <div class="bg-green-2 p-small radius-normal h-full">Stretches to the full height of the row</div>
    </div>
  </div>
`}
/>

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:align-content-start`, to apply the corresponding responsive class.

<CssResponsiveTable all />

<Code
  language="html"
  code={`
<div class="grid is-mobile">
    <div class="col mobile:is-three-quarters tablet:is-two-thirds desktop:is-half widescreen:is-one-third fullhd:is-one-quarter">
      <div class="bg-green p-small radius-normal">
        is-three-quarters-mobile<br>
        is-two-thirds-tablet<br>
        is-half-desktop<br>
        is-one-third-widescreen<br>
        is-one-quarter-fullhd
      </div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">auto</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">auto</div>
    </div>
    <div class="col">
      <div class="bg-green p-small radius-normal">auto</div>
    </div>
  </div>
`}
/>
