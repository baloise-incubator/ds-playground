## Integration

The `step` option can also be specified using a property.
To facilitate this, use the utility function called `newBalStepOption`.

```typescript
import { newBalStepOption } form '@baloise/ds-core'

const options = [
  newBalStepOption({ label: "My Label 1", value: "1" }),
  newBalStepOption({ label: "My Label 2", value: "2" }),
]

// next pass the option array to the component ...

```
