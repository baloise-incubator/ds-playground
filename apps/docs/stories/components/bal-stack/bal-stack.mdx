<Lead>The **Stack** element is responsible for arranging its direct descendants in a vertical or horizontal direction,
  while also allowing for the inclusion of optional spacing between each child.</Lead>

<InfoQuote>
**TIP**

The **Stack** component deals with layouts in one dimension, whereas the [Grid](?path=/docs/foundation-grid--documentation) manages layouts in two dimensions.
By default, the **Stack** component stacks children vertically in a column.
</InfoQuote>

The default arrangement for Stack is a vertical column. Nevertheless, the direction prop allows items to be positioned horizontally in a row as well.
