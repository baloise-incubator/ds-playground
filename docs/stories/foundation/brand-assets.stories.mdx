## Logo

The Baloise logo only appears in blue (positive) and white (negative). In all digital touch points,
the Baloise logo and shapes appear in their full dynamic and beauty. There are two logo animations available:
a short version for website and application design, and an extended animation for all AV media applications.

## Favicons

To ensure logos remain clearly visible even on small touch points, the Baloise Shape, a expression of the logo,
is displayed on a blue background. Additionally, colored backgrounds and a negative version of the logo are permissible.

## Shapes

On digital touchpoint use the animated version of the Baloise Shapes to convey a modern and dynamic approach to the brand.

import ShapeSizes from '../assets/images/foundations/brand-assets/shape-sizes.png'

### Sizes

There are 3 different shape sizes:

- Desktop
- Tablet
- Mobile

Each shape can be rotated in 90°, clockwise steps.

<ImageCanvas>
  <img src={ShapeSizes} />
</ImageCanvas>

import ShapeTypes from '../assets/images/foundations/brand-assets/shape-types.png'

### Types

We have 3 different type of shapes that can be randomly combined with one of the four Baloise primary colors (only
use Tint 3 of each color). The type of shape can be selected randomly and is not attached to any color.

<ImageCanvas>
  <img src={ShapeTypes} />
</ImageCanvas>

import ShapePosition from '../assets/images/foundations/brand-assets/shape-position.png'

### Position

The Baloise Shape should be used in the header area and only once per page as a maximum.

  <img src={ShapePosition} />
</ImageCanvas>
