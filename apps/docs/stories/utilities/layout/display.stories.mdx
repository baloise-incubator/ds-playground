## Classes

<CssLayoutDisplay />

## Hidden

An element that is hidden is not visible and does not occupy any space.

<Code
  language="html"
  code={`
<div class="flex gap-normal">
  <div class="hidden bg-green p-normal radius-normal">1</div>
  <div class="bg-green p-normal radius-normal">2</div>
  <div class="bg-green p-normal radius-normal">3</div>
</div>
`}
/>

## Block

A block element begins on a new line.

<Code
  language="html"
  code={`
<div class="block bg-green text-align-center p-normal radius-normal mb-normal">1</div>
<div class="block bg-green text-align-center p-normal radius-normal mb-normal">2</div>
<div class="block bg-green text-align-center p-normal radius-normal mb-normal">3</div>
`}
/>

## Inline

An inline element does not initiate on a new line and takes up width as required.

<Code
  language="html"
  code={`
<div class="inline bg-green text-align-center p-normal radius-normal mx-normal">1</div>
<div class="inline bg-green text-align-center p-normal radius-normal">2</div>
<div class="inline bg-green text-align-center p-normal radius-normal mx-normal">3</div>
`}
/>

## Inline Block

Inline block behaves much like an inline element, but it also respects properties such as width, height, and top/bottom paddings/margins.

<Code
  language="html"
  code={`
<div class="inline-block bg-green text-align-center p-normal radius-normal">1</div>
<div class="inline-block bg-green text-align-center p-normal radius-normal mx-normal">2</div>
<div class="inline-block bg-green text-align-center p-normal radius-normal">3</div>
`}
/>

## Flex

Sets the element to display as a block-level flex container.

<Code
  language="html"
  code={`
<div class="flex gap-normal">
    <div class="flex-1 bg-green text-align-center p-normal radius-normal">1</div>
    <div class="flex-1 bg-green text-align-center p-normal radius-normal">2</div>
    <div class="flex-1 bg-green text-align-center p-normal radius-normal">3</div>
</div>
`}
/>

## Inline Flex

Sets the element to display as an inline-level flex container.

<Code
  language="html"
  code={`
<div class="inline-flex gap-normal">
    <div class="flex-1 bg-green text-align-center p-normal radius-normal">1</div>
    <div class="flex-1 bg-green text-align-center p-normal radius-normal">2</div>
    <div class="flex-1 bg-green text-align-center p-normal radius-normal">3</div>
</div>
`}
/>

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:hidden`, to apply the corresponding responsive class.

<CssResponsiveTable all />

<Code
  language="html"
  code={`
<div class="bg-yellow-2 p-small mb-small mobile:hidden">hidden-mobile</div>
<div class="bg-yellow-3 p-small mb-small tablet:hidden">hidden-tablet</div>
<div class="bg-yellow-4 p-small mb-small touch:hidden">hidden-touch</div>
<div class="bg-red-2 p-small mb-small desktop:hidden">hidden-desktop</div>
<div class="bg-red-3 p-small mb-small high-definition:hidden">hidden-high-definition</div>
<div class="bg-red-4 p-small mb-small widescreen:hidden">hidden-widescreen</div>
<div class="bg-red-5 p-small mb-small fullhd:hidden">hidden-fullhd</div>
<hr>
<div class="bg-purple-2 p-small mb-small tablet-only:hidden">hidden-tablet-only</div>
<div class="bg-purple-3 p-small mb-small desktop-only:hidden">hidden-desktop-only</div>
<div class="bg-purple-4 p-small mb-small high-definition-only:hidden">high-definition-only</div>
<div class="bg-purple-5 p-small mb-small widescreen-only:hidden">hidden-widescreen-only</div>
`}
/>
