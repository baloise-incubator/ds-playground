## Classes

<CssTypographyWeight />

## Bold

Both font families offer a bold style option.

<Code
  language="html"
  code={`
<h1 class="font-weight-bold title">Bold - Title</h1>
<p class="font-weight-bold">Bold - Lorem ipsum dolor sit, amet consectetur adipisicing elit.</p>
`}
/>

## Regular

Only the **text** font families, which is default, offer a regular style option.

<Code
  language="html"
  code={`
<p>Regular - Lorem ipsum dolor sit, amet consectetur adipisicing elit.</p>
`}
/>

## Light

Only the **title** font families offer a light style option.

<Code
  language="html"
  code={`
<h1 class="text-weight-light title">Light - Title</h1>
`}
/>
