### bal-label
 
#### Properties

| Property    | Attribute   | Description                                                                                                                                                                                                                    | Type                                                                           | Default     |
| ----------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------ | ----------- |
| `disabled`  | `disabled`  | If `true`, the element is not mutable, focusable, or even submitted with the form.
