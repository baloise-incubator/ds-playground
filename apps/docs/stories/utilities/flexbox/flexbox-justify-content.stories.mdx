## Classes

<CssFlexboxJustifyContent />

## Start

Elements are aligned to the start of the container.

<Code
  language="html"
  code={`
<div class="flex justify-content-start flex-flex-wrap gap-normal">
    <div class="bg-green radius-normal p-normal">1</div>
    <div class="bg-green radius-normal p-normal">2</div>
    <div class="bg-green radius-normal p-normal">3</div>
</div>
`}
/>


## Center

Elements are centered within the container.

<Code
  language="html"
  code={`
<div class="flex justify-content-center flex-flex-wrap gap-normal">
    <div class="bg-green radius-normal p-normal">1</div>
    <div class="bg-green radius-normal p-normal">2</div>
    <div class="bg-green radius-normal p-normal">3</div>
</div>
`}
/>

## End

Elements are aligned to the end of the container.

<Code
  language="html"
  code={`
<div class="flex justify-content-end flex-flex-wrap gap-normal">
    <div class="bg-green radius-normal p-normal">1</div>
    <div class="bg-green radius-normal p-normal">2</div>
    <div class="bg-green radius-normal p-normal">3</div>
</div>
`}
/>

## Space Between

Elements are spaced evenly between them.

<Code
  language="html"
  code={`
<div class="flex justify-content-space-between flex-flex-wrap gap-normal">
    <div class="bg-green radius-normal p-normal">1</div>
    <div class="bg-green radius-normal p-normal">2</div>
    <div class="bg-green radius-normal p-normal">3</div>
</div>
`}
/>

## Space Around

Elements are spaced evenly between and around them.

<Code
  language="html"
  code={`
<div class="flex justify-content-space-around flex-flex-wrap gap-normal">
    <div class="bg-green radius-normal p-normal">1</div>
    <div class="bg-green radius-normal p-normal">2</div>
    <div class="bg-green radius-normal p-normal">3</div>
</div>
`}
/>

## Space Evenly

Elements are spaced evenly around them.

<Code
  language="html"
  code={`
<div class="flex justify-content-space-evenly flex-flex-wrap gap-normal">
    <div class="bg-green radius-normal p-normal">1</div>
    <div class="bg-green radius-normal p-normal">2</div>
    <div class="bg-green radius-normal p-normal">3</div>
</div>
`}
/>

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:justify-content-start`, to apply the corresponding responsive class.
