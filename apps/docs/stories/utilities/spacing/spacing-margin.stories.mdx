## Classes

Spacing can be set with **margin** `m*` for all directions:

- `*t` for **top**
- `*r` for **right**
- `*b` for **bottom**
- `*l` for **left**
- `*x` **horizontally** for both left and right
- `*y` **vertically** for both top and bottom

<CssSpacingMargin />

## Single Side Margin

Margin can be defined for a specific edge.

<Code
  language="html"
  code={`
<div class="flex flex-wrap align-items-center justify-content-center gap-large">
  <div class="bg-red-2">
      <div class="mt-normal bg-green p-large">mt-normal</div>
  </div>
  <div class="bg-red-2">
      <div class="mr-large bg-green p-large">mr-large</div>
  </div>
  <div class="bg-red-2">
      <div class="mb-small bg-green p-large">mb-small</div>
  </div>
  <div class="bg-red-2">
      <div class="ml-medium bg-green p-large">ml-medium</div>
  </div>
</div>
`}
/>

## Horizontal Margin

Shorthand classes allow you to define the same margin value for both the left and right sides.

<Code
  language="html"
  code={`
<div class="flex flex-wrap align-items-center justify-content-center gap-large">
    <div class="bg-red-2 radius-normal">
        <div class="mx-large radius-normal bg-green font-weight-bold p-normal">mx-large</div>
    </div>
</div>
`}
/>

## Vertical Margin

The same margin value can also be defined at the top and bottom sides using shorthand classes.

<Code
  language="html"
  code={`
<div class="flex flex-wrap align-items-center justify-content-center gap-large">
    <div class="bg-red-2 radius-normal">
        <div class="my-large radius-normal bg-green font-weight-bold p-normal">my-large</div>
    </div>
</div>
`}
/>

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:m-large`, to apply the corresponding responsive class.
